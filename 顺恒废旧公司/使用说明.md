# 怎么用这个工具

## 装东西（只要装一次）

电脑上要有 Python。打开终端（cmd 或者 PowerShell），输入：

```
pip install openpyxl xlrd
```

## 跑报表（每次三步）

### 第1步：把文件放好

把下面这些文件扔到 `顺恒废旧公司` 这个文件夹里：

- 期初资产负债表（去年12月的）
- 销售收入数据（1-5月的那个 Excel）
- 成本费用数据（1-5月的那个 Excel）
- 工资表（每个月一个）
- 银行流水（农行和信用社，每个月各一个）

> 文件名不用改，就用原来的名字就行。

### 第2步：运行

打开终端，进入到这个文件夹，输入：

```
python generate_reports.py
```

等着它跑完，大概几秒钟。

### 第3步：拿结果

打开 `报表输出` 文件夹，里面有两个文件：

- **Excel 文件** — 三张表都在里面（利润表、资产负债表、现金流量表）
- **说明文件** — 告诉你有啥问题需要注意

## 如果想换月份

打开 `generate_reports.py`，改最上面这几行：

```python
REPORT_START = date(2026, 1, 1)   # 改成你要的开始日期
REPORT_END = date(2026, 5, 31)    # 改成你要的结束日期
MONTHS = [1, 2, 3, 4, 5]          # 改成你要的月份
```

然后把下面的文件名也改成对应的月份，比如把 `1月` 改成 `6月`。

## 跑完看看对不对

跑完之后终端会打印结果：

- 看到全是 `[PASS]` — 没问题
- 看到 `[WARN]` — 有点小问题，打开说明文件看看
- 看到 `[FAIL]` — 数据可能有问题，检查一下文件
