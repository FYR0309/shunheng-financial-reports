# 顺恒废旧公司 2026年1-5月财务报表生成 — 设计文档

## 概述

为来宾市顺恒废旧物质回收有限公司生成 2026年1-5月财务报表（利润表、资产负债表、现金流量表），基于已有的 21 个源文件（2025年12月末报表 + 2026年1-5月发票/银行流水/工资数据），沿用《小企业会计准则》格式。

## 源文件清单

| 类别 | 文件 | 用途 |
|---|---|---|
| 2025年末报表 | 利润表、现金流量表、余额表、资产负债表（往来重分类） | 提取期初数 |
| 销售收入 | 顺恒2026年1-5月销售收入数据.xlsx | 发票明细 → 营业收入 |
| 成本费用 | 顺恒2026年1-5月成本费用数据.xlsx | 采购发票 → 营业成本、管理费用 |
| 银行流水 | 农行明细 ×5 + 农村信用社明细 ×5 | 现金流量表、银行存款余额 |
| 工资薪金 | 工资薪金 ×5 | 管理费用-工资、应付职工薪酬 |

## 架构

```
源文件 (21个) → generate_reports.py → 输出/
                                        ├── 顺恒2026年1-5月财务报表.xlsx
                                        └── 顺恒2026年1-5月报表说明.md
```

单一 Python 脚本，无外部服务依赖。仅使用 `openpyxl` 和 `xlrd`（已验证可用）。

## 模块设计

### 1. 数据读取模块
- 支持 `.xls` 和 `.xlsx` 双格式
- 按文件命名规则识别类型（收入/成本/银行/工资/报表）
- 读取失败的文件记录到错误日志，不中断流程

### 2. 利润表生成
- 营业收入：按发票日期筛选1-5月，汇总不含税金额（收入发票"信息总表"sheet）
- 营业成本：按采购发票汇总不含税金额
- 税金及附加：增值税实缴汇总 + 附加税按增值税比例估算
- 管理费用：工资（5个月实发 + 社保单位部分）+ 折旧（固定资产原值约479万，直线法）+ 租金（从成本发票租赁类提取）+ 银行手续费
- 所得税：小微企业优惠税率估算
- 输出：1-5月累计 + 分月明细

### 3. 资产负债表生成
- 期初数全部来自 2025年12月资产负债表
- 期末数（2026年5月31日）通过期初 + 本期发生额计算
- 货币资金 = 期初 + 两银行流水净额（已验证末笔余额）
- 存货简化处理为：期初 + 采购总额 - 销售成本
- 固定资产 = 期初净值 - 5个月折旧
- 未分配利润 = 期初 + 本期净利润
- 其他应付款 664万默认不变，如有银行流水中关联方往来则标注
- 长期待摊费用在说明文件中询问是否摊销

### 4. 现金流量表生成
- 合并两银行账户流水
- 按对方名称/摘要自动分类：
  - 销售收现：个人大额转入
  - 采购付现：对公转出
  - 支付工资：员工姓名 + 固定金额模式
  - 支付税费：税局/银行扣费
  - 其他/待确认：无法确定的条目

### 5. 输出模块
- 三张报表各一个 sheet，写入一个 `.xlsx` 文件
- 按小企业会计准则行次格式
- 一份 `.md` 说明文件，包含：
  - 关键发现
  - 异常/不确定事项
  - 校核结果
  - 待人工确认条目

## 数据流

```
期初余额表 (2025.12) ──────────────────┐
                                       ├──→ 资产负债表
销售收入发票 (1-5月) ──→ 收入汇总 ────┤
成本费用发票 (1-5月) ──→ 成本汇总 ────┤──→ 利润表
工资数据 (1-5月) ──────→ 工资汇总 ────┤
                                       │
银行流水 (1-5月, 两户) ────────────────┴──→ 现金流量表
```

## 验证策略

脚本自动校核：
1. **资产负债表平衡**：资产总计 == 负债 + 所有者权益
2. **表间勾稽**：本期净利润 == 未分配利润变动额
3. **银行余额核对**：脚本算出余额 vs 流水末笔余额
4. **发票总额交叉校验**：发票汇总 vs 报表科目数

校核结果写入说明文件。

## 异常处理

| 情况 | 处理 |
|---|---|
| 发票与银行流水不匹配 | 不强制匹配，报表以发票为准，差异标注 |
| 银行流水无法自动分类 | 归入"待确认"，列出明细 |
| 源文件格式不一致 | 兼容 xls/xlsx，失败文件记录错误日志 |
| 数据日期超出 1-5 月范围 | 自动按日期筛选，跨期数据标注 |
